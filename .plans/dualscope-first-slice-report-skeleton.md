# dualscope-first-slice-report-skeleton

## Background

The first-slice smoke artifacts passed artifact validation. This final stage turns the validated first-slice setting into a paper-facing report skeleton.

## Goal

Create a concise report skeleton that records setting, expected metrics, table placeholders, artifact links, limitations, and the next experiment recommendation.

## Non-goals

- No full paper writing.
- No performance claims beyond smoke validation.
- No full experiment matrix.
- No route_c continuation.

## Required content

- first slice setting
- dataset/model/trigger/target/capability configuration
- baseline inclusion
- expected metrics
- expected tables
- artifact links
- known limitations
- next experiment recommendation

## Milestones

- M1: report skeleton scope frozen
- M2: report / table / figure placeholders completed
- M3: single verdict and single recommendation completed

## Exit criteria

The report skeleton is validated if it exists, is readable, links the first-slice artifacts, and does not claim full experimental results.
