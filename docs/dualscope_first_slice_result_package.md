# DualScope First-Slice Result Package

The result package summarizes what the current first-slice can support.

It can support artifact-chain reporting, schema compatibility, local model generation probe status, local logits-derived probability probe status, and cost/decision output shapes.

It cannot yet support AUROC, F1, ASR, clean utility, or final paper-level performance claims because performance labels and full Stage 1/Stage 2 model-integrated execution are not yet available.
