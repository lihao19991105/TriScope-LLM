# DualScope Real-Run Preflight Checklist

- Dataset path exists and has the expected JSONL schema.
- Model path exists and tokenizer can load.
- GPU availability is confirmed.
- Output directory is writable.
- Stage 1, Stage 2, and Stage 3 contract artifacts exist.
- Capability mode is selected and fallback policy is explicit.
- Disk space is sufficient.
- Python modules compile.
- Dry-run config references resolve.
